# LinkedIn Clone Backend

A RESTful backend API for a LinkedIn Clone built with **Node.js**, **Express.js**, and **MongoDB**. The application follows the **MVC architecture** and provides APIs for user authentication, profile management, resume handling, and PDF resume generation.

## Features

* User Registration & Login
* Secure Password Hashing using Bcrypt
* User Profile Management
* Profile Picture Upload using Multer
* Resume Upload & Download
* Dynamic PDF Resume Generation using PDFKit
* MongoDB Integration with Mongoose
* RESTful APIs following MVC Architecture

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* Bcrypt
* Multer
* PDFKit

## Installation

```bash
git clone <repository-url>
cd Backend
npm install
```

Create a `.env` file:

```env
MONGO_URI=<your_mongodb_connection_string>
PORT=5000
```

Start the server:

```bash
npm run dev
```
